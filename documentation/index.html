Simple Documentation.

This is where you can find documentation on the "sentences" used by Simple.

These are just like commands in other languages.


  • Create a(n) [Integer|Boolean|Float|String], [variable name], with a starting value of [value].

    • This creates a variable, 
      [variable name]
      , which is a Boolean (True or False), Integer (whole numbers only), String (a list of characters/text), or Float (number with decimals). The value of the variable is in 
      [value]
      , which must be a value of the same type as the variable itself. (ex. True if the variable is a Boolean, 5 if the variable is an Integer, "Hello" if the variable is a String, or 5.5 if the variable is a Float.)
    • [value]
      cannot be an expression. It must be a definite value.
  • Set variable [variable] to [value].
    • Sets the value stored in 
      [variable]
      to 
      [value]
      .
    • This can be an expression that uses other variables and performs calculations.
  • Print the value of [variable] to the console.
    • Does exactly what it says on the tin. It prints the value stored in 
      [variable]
      to the console.
  • Increment the value of [variable].
    • Adds one to the value of 
      [variable]
      .
  • Decrement the value of [variable].
    • Subtracts one from the value of 
      [variable]
      .
  • Alert the user with the value of [variable].
    • Sends an alert to the user with the contents of 
      [variable]
      .
  • Until [condition]: 
    - [code]
    • Until 
      [condition]
      is 
      true
      , 
      [code]
      will be executed.
    • Multiple lines of code in the loop are supported, but each line of code has to start with 
      -
      , or it is outside the loop, and will be executed as such.
  • Append the character '[character]' to the string stored in [string variable]
    • Adds 
      [character]
      to the end of the string stored in 
      [string variable]
      .
    • Only one character can be added at a time.
  • Prepend the character '[character]' to the string stored in [string variable]
    • Adds 
      [character]
      to the beginning of the string stored in 
      [string variable]
      .
    • Only one character can be added at a time.
  • Remove the character at position [index] of the string stored in [string variable]
    • Removes the character found at 
      [index]
      in the string specified in 
      [string variable]
      .
    • Only one character can be removed at a time.
    • Note that the first position in the string is 1, the second is 2, the third 3, etc. In some languages (not this one), the first position is 0, the second is 1, the third 2, etc.

  • Comments:

    • Comments are entirely ignored by the interpreter. You can use them to explain your code, remove dysfunctional code, and more.
    • //[comment]
      (single-line comment):
      • When placed at the beginning of a line, the characters 
        //
        mark that line as a comment.
    • #[comment]#
      (Multi-line comment):
      • No matter how many lines of anything are between the two hash marks, it will all be ignored by the interpreter.
