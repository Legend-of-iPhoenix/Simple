Simple Documentation.

This is where you can find documentation on the "sentences" used by Simple.

These are just like commands in other languages.

 

  • Create a(n) [Integer|Boolean|Float], [variable name], with a starting value of [value].
    • This creates a variable, [variable name], which is a Boolean (True or False), Integer (whole numbers only), or Float (number with decimals). The value of the variable is in [value], which must be a value of the same type as the variable itself. (i.e. "True" if the variable is a Boolean, 5 if the variable is an Integer, or 5.5 if the variable is a Float.)
    • [value] cannot be an expression. It must be a definite value.
  • Set variable [variable] to [value].
    • Sets the value stored in [variable] to [value].
    • This can be an expression that uses other variables and performs calculations.
  • Print the value of [variable] to the console.
    • Does exactly what it says on the tin. It prints the value stored in [variable] to the console.
  • Increment the value of [variable].
    • Adds one to the value of [variable].
  • Decrement the value of [variable].
    • Subtracts one from the value of [variable].
